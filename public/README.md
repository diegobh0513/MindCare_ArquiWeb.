# MindCare Landing Page

## 📋 Descripción
Landing page profesional para **MindCare**, una plataforma tecnológica diseñada para el cuidado de la salud mental en adultos jóvenes y universitarios mediante Machine Learning simulado.

## 🎯 Características Principales

- **Header Sticky**: Navegación fija con enlaces rápidos a todas las secciones
- **Hero Section**: Propuesta de valor clara y llamada a la acción
- **Sección de Problema**: Estadísticas y contexto sobre la crisis de salud mental
- **Solución**: Explicación del concepto y beneficios de MindCare
- **Características**: 6 características principales con iconos y descripciones
- **Cómo Funciona**: Explicación visual del proceso de ML simulado en 4 pasos
- **Impacto**: Valores y compromisos de MindCare
- **CTA**: Formulario de suscripción a beta
- **Footer**: Enlaces y información de contacto

## 🎨 Diseño

- **Estilo**: Profesional y corporativo
- **Colores**: Azul (#2563eb) como color principal
- **Tipografía**: Segoe UI / Arial (sans-serif)
- **Responsive**: Totalmente adaptado para móvil, tablet y desktop

## 🚀 Uso

1. Abre `index.html` en tu navegador
2. La página es totalmente funcional sin necesidad de servidor
3. El formulario de suscripción guarda emails en `localStorage` (para demo)

## 🔧 Funcionalidades

### Navegación
- Links suave entre secciones
- Header sticky para acceso rápido
- Botones funcionales de CTA

### Formulario de Beta
- Validación de email en tiempo real
- Almacenamiento local de suscriptores
- Mensajes de confirmación

### Animaciones
- Fade-in al scroll de elementos
- Hover effects en botones y tarjetas
- Transiciones suaves

## 📱 Breakpoints Responsivos
